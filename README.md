<h1>Projeto para Secretaria de Cursinho Popular :large_blue_circle::green_circle::yellow_circle::red_circle:</h1>

<h2>Escopo</h2>

<p>Esse projeto tem por finalidade ler uma planilha que registra a presença dos alunos e mandar uma mensagem automatizada pelo whatsapp caso o aluno chegue a ter 4 ou mais faltas consecutivas.</p>

<h3>Ferramentas</h3>

- <a href="https://azure.microsoft.com/pt-br/products/devops">Azure DevOps</a>;
- <a href="https://www.python.org/">Python</a>;
- <a href="https://www.djangoproject.com/">Django</a>;
- <a href="https://pandas.pydata.org/">Biblioteca Pandas</a>;
- <a href="https://docs.google.com/spreadsheets/create?hl=pt-br">Google Sheets</a>;
- <a href="https://github.com/">Github</a>;
